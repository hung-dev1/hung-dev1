## 📊 Projects

### 📈 Stock Market Dashboard
A modern, responsive front-end application for tracking and analyzing stock market data in real-time.

**Repository:** [stock-market-dashboard](https://github.com/hung-dev1/stock-market-dashboard)

**Features:**
- Real-Time Stock Quotes tracking live stock prices and market movements
- Interactive Charts with candlestick and line charts visualization
- Portfolio Management with watchlist functionality
- Market Analysis Tools with technical indicators
- Responsive Design for desktop, tablet, and mobile devices
- Dark Mode Support with light and dark themes

**Tech Stack:**
- Frontend: React.js with Vite
- UI: Tailwind CSS / Material-UI
- State Management: Redux / Context API
- Charting: Chart.js / ECharts
- HTTP Client: Axios
- APIs: Alpha Vantage, IEX Cloud, Finnhub

**Status:** 🟢 In Development | Educational Project
